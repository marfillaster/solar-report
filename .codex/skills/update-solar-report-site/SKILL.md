# Update Solar Report Site

Regenerate the public site with the bundled script instead of hand-editing the generated pages.

## Workflow

1. Confirm the source report path.
   Default source: `/Users/ken/solar-skills/data/solar-analysis.md`

2. Run the generator from the repo root:

```bash
python3 .codex/skills/update-solar-report-site/scripts/update_site.py
```

Optional flags:

```bash
python3 .codex/skills/update-solar-report-site/scripts/update_site.py \
  --source /absolute/path/to/solar-analysis.md \
  --repo-root /absolute/path/to/solar-report
```

3. Review the diff for:
   - `index.html`
   - `full-report.html`

4. If the user asked to publish, commit and push the repo after reviewing the generated changes.

## Deterministic Rules

- Treat `.codex/skills/update-solar-report-site/scripts/update_site.py` as the source of truth for generated page content.
- Do not hand-edit `index.html` or `full-report.html` to change report content unless you are intentionally updating the generator logic.
- Redact the public location to `Cavite, Philippines` on generated pages.
- If required report sections are missing, stop and report the missing headings instead of improvising replacements.

## Extraction Contract

The generator expects these report sections:

- `Executive Summary`
- `System Profile`
- `Alerts`
- `Recommendations`
- `Bill Impact`
- `ROI Estimate`
- `Battery Health`
- `Annual Projection`

The summary page is built from fixed extractions:

- hero/system chips from `System Profile`
- summary paragraph from `Executive Summary`
- payback from `ROI Estimate`
- annual bill reduction from `Bill Impact`
- annual generation and carbon impact from `Annual Projection`
- top recommendation from the first numbered recommendation subsection
- alerts table from `Alerts`
- battery health bullets from `Battery Health`
- monthly bill table from `Bill Impact`

The full report page is generated by converting the full markdown report into constrained HTML with headings, paragraphs, lists, and tables.
The raw source report is also published as `solar-analysis.md` in the repo root.
